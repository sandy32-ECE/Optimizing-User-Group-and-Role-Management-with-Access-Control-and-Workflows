# Optimizing User, Group and Role Management with Access Control and Workflows

## Project Overview
This project demonstrates the implementation of user, group, and role management in ServiceNow using Access Control Lists (ACLs) and Flow Designer automation.

## Features
- Created users, groups, and roles
- Configured Task Table 2
- Implemented ACLs
- Built an approval workflow using Flow Designer
- Tested and validated the workflow

## Technologies Used
- ServiceNow
- Flow Designer
- Access Control Lists (ACLs)
- GitHub

## Outcome
Successfully implemented and tested user management, access control, and workflow automation in ServiceNow.
